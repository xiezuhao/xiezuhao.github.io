部署说明：
1. 将本文件夹内所有文件上传到你的 GitHub 仓库 xiezuhao.github.io 的根目录。
2. 保持目录结构：
   - index.html
   - assets/style.css
   - assets/app.js
   - assets/data.js
3. 提交后等待 GitHub Pages 自动重新构建。
4. 访问：
   - https://xiezuhao.github.io
   - https://www.yanglaoziyuan.top

如需替换为你正式 ARIMA 结果，请编辑 assets/data.js 中的 forecast 字段。
